# PYTE
## The Program for To-Do List Outlining and Note-taking

### Advanages:
- Tasks
- Notes
- Calendar
- Completion diagram
- Hierarchical to do lists
- Filtering the tasks and notes
- Possibility to add mini-notes to every task
- Cross-linked and tagged data
- Advanced text editor
